# Portfolio – Sascha Schumbera

Personal portfolio website built with Next.js, showcasing projects, skills, and experience.

**Live:** [sascha-schumbera.dev](https://sascha-schumbera.dev) 

## Tech Stack

- **Framework:** Next.js 16 (Turbopack)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Deployment:** Vercel

## Features

- Project showcase with interactive demo videos
- Keyword-based chatbot for visitor questions
- Dark/light mode
- Responsive design

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
